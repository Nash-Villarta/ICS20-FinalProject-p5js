The p5js Art Maker!








Hello! This is my art program using shapes using p5js. To get started, try drawing on the draw space with left click. To delete all, use Right Click. Above is the input to change the circles that you draw. By using circles, you can achieve anything you wan't to draw! Having a low mouse sensitivity allows for less spaces in between each circle. Enjoy!


Controls Function
Left Click Create Circle
HOLD Left Click Create Circles
Right Click Delete All Shapes
Up Arrow Generate Draw Idea
